# Student Marks Manager

A simple Python CLI application to manage student records with persistent JSON storage.

## 🚀 Features
- Add, update, and delete student records
- View all student data
- Calculate:
  - Highest and lowest scorer
  - Average marks
  - Pass/Fail results and percentages
- Data stored using JSON (persistent across runs)
- Input validation and error handling

## 🛠️ Tech Stack
- Python
- JSON (for storage)

## ▶️ How to Run
```bash
python main.py
```

**📌 Notes**

Marks range: 0–100

Passing marks: 50
